About Air Quality Monitoring Dashboard

The Air Quality Monitoring Dashboard is a web application designed to provide real-time environmental data including temperature, humidity, and air quality levels. By leveraging public APIs, the dashboard helps users monitor and understand their surrounding environment.

The data is sourced from trusted public APIs such as the Korea Meteorological Administration and AirKorea.

Key Features

1. Real-time Data Visualization Visualize temperature, humidity, and air pollutant levels using interactive charts.

2. User-friendly Interface A clean and intuitive layout ensures easy navigation and quick access to data.

3. Interactive Data Filtering Users can select specific cities and districts to view localized environmental data.

4. Detailed Pollutant Analysis Displays individual air pollutant concentrations such as PM2.5, PM10, NO₂, SO₂, O₃, and CO.

You can access the full source code of this website on my GitHub repository. The repository contains all implementation details, including HTML, CSS, and JavaScript.
